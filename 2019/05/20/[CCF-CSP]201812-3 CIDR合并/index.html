CIDR合并 2.576KB C++ 正确 100 171ms 3.906MB

关键点：

  1. 用C的sscanf()巧妙处理字符串
  2. 将IP地址由点分十进制转换为32位二进制，便于之后的合并处理
  3. 将前缀转为二进制掩码，直接和IP的32位二进制相“与”。
  4. 本题含有大量删除操作，应用list处理ip列表。如果用顺序表会超时，最多得90分。
